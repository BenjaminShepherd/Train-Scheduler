# Train Scheduler

Tracks trains frequency, next arrival, & minutes away. Users can add new trains.
Uses firebase to store train information, & Moment.js to track the trains.

## Getting Started


### Installing

npm install 

### Executing program


```
npm start
```
### Technology Used

HTML/CSS, jQuery, JavaScript, Moment.js, Firebase
